05 — Iconography

Icon library: Lucide

One icon family across every surface. Lucide is MIT-licensed, 1.5-2px stroke, 1400+ icons, and matches the Inter / slate aesthetic. Standard size 20px (UI), 24px (feature), 16px (dense rows). Stroke = currentColor so icons auto-adapt to theme.
